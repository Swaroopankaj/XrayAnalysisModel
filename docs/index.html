X-ray Analysis

A Deep Learning Project for Identifying and Classifying Forearm Conditions from X-ray Images

Project Overview

This project develops an intelligent system to assist in the analysis of forearm X-ray images. Utilizing advanced deep learning techniques, the model is trained to accurately identify and classify various forearm conditions, distinguishing between normal and abnormal findings. Our approach prioritizes robust performance and efficient processing, making it a valuable tool for healthcare applications.

Key Features

Automated Dataset Organization

Scripted preprocessing and structuring of raw medical image datasets (e.g., MURA) into standardized training, validation, and test sets.

Deep Learning Model Training

Utilizes a pre-trained ResNet50 CNN with transfer learning, configurable parameters, and GPU acceleration for efficient and accurate model training.

Comprehensive Model Evaluation

Provides detailed performance metrics including accuracy, classification reports, confusion matrices, and ROC curves for thorough model assessment.

Efficient Model Inference

A dedicated script for quick and robust predictions on new, unseen X-ray images, indicating the classified condition.

How to Get Started

To set up and run this project, please refer to the detailed instructions in the README.md file in the project's GitHub repository.

It covers prerequisites, dataset acquisition (using `AzCopy` for MURA), data organization, model training, and inference.
